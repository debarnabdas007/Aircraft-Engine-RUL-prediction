About AEGIS AERO

AEGIS AERO is an end-to-end Machine Learning web application designed to solve a critical problem in the aviation industry: Predictive Maintenance.

The Problem

Aircraft engines degrade over time. If an airline replaces parts too early, they waste millions of dollars. If they wait too long, it results in catastrophic failure. The goal is to predict the exact Remaining Useful Life (RUL) of an engine based on live sensor readings.

The Data & AI Architecture

This project uses the famous NASA Turbofan Engine Degradation Simulation dataset. The intelligence behind this application includes:

  • Advanced Feature Engineering: Converting raw sensor data into 5-cycle rolling averages and standard deviations to capture the degradation trend over time.
  • Automated MLOps Pipeline: A fully modular architecture including Data Ingestion, Transformation, and Model Training components.
  • Best Model: A Random Forest Regressor optimized to catch complex non-linear sensor patterns.

How it Works

A mechanic enters the current readings from 10 crucial engine sensors (like temperature, pressure, and bypass ratio) into the web form. The Prediction Pipeline dynamically generates the required time-series features in a steady state and feeds them to the frozen AI model to generate an instant RUL prediction.
